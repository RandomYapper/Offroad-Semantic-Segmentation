## 🎯 Overview

Autonomous navigation in off-road environments presents unique challenges compared to urban driving. This project develops a high-performance semantic segmentation model for **Unmanned Ground Vehicles (UGVs)** to enable safe navigation in complex, unstructured desert terrains.

By leveraging an ensemble of foundation models (**DINOv2** and **SegFormer**) and implementing novel class-remapping strategies, we achieve a **Mean IoU of 0.5390** on synthetic data generated by Duality AI's Falcon platform.

### Why This Matters

- **Safety-Critical**: Enables UGVs to distinguish traversable terrain from obstacles
- **Synthetic Data Advantage**: Trained on cost-effective, pixel-perfect synthetic data
- **Real-World Ready**: Designed for transfer to physical deployment scenarios
- **Edge Case Coverage**: Handles rare weather events and lighting conditions

---

## ✨ Key Features

- 🤖 **Dual-Model Ensemble**: Combines DINOv2 (ViT-B/14) for semantic robustness and SegFormer (MiT-B4) for fine-grained detail
- 🧠 **Learnable Unifier**: Novel convolutional mixer that optimally fuses predictions from both models
- 🎯 **Smart Class Remapping**: Handles label ambiguity by merging semantically similar classes
- 📊 **Mean IoU: 0.5390**: Strong performance on challenging desert terrain
- 🌟 **Near-Perfect Sky Segmentation**: 0.9793 IoU on Sky class
- 💪 **Robust to Lighting**: Extensive photometric augmentation for shadow resistance

## 🏗️ Architecture

### Dual-Stream Ensemble Design

Our solution employs a parallel architecture that processes images through two complementary pathways:

```
Input Image (RGB)
     │
     ├─────────────────┬
     │                 │                 
   DINOv2          SegFormer         
(ViT-B/14)        (MiT-B4)         
     │                 │                
Multi-Scale       All-MLP          
UNet Decoder      Decoder          
     │                 │                
Logits (10)      Logits (10)      
     │                 │                
     └─────────┬───────┘                
               │                        
      Learnable Unifier                
      (1×1 Conv Mixer)                 
               │                        
      Final Segmentation Map            
```

### Model Components

#### 1. DINOv2 (ViT-B/14)
- **Purpose**: Semantic robustness and global context
- **Backbone**: Vision Transformer with self-supervised pre-training
- **Decoder**: Custom Multi-Scale UNet with skip connections
- **Strengths**: Excellent for "Sky" and "Landscape" classes

#### 2. SegFormer (MiT-B4)
- **Purpose**: Fine-grained detail and texture discrimination
- **Backbone**: Mix Transformer (hierarchical feature pyramid)
- **Decoder**: Lightweight All-MLP decoder
- **Strengths**: Superior for small objects like "Logs" and "Ground Clutter"

#### 3. Learnable Unifier
- **Type**: 1×1 Convolutional Layer
- **Function**: Per-pixel logistic regression on concatenated logits
- **Advantage**: Learns class-specific model weighting automatically

## 💻 Usage

### Quick Inference

```python

from utils import load_image, visualize_prediction
import torch

# Load the ensemble model

    dino_weights='checkpoints/dinov2_best.pth',
```

## 🎓 Model Training

### Training DINOv2

```bash
python train_dinov2.py \
    --data_dir ./data/desert_dataset/ \
    --batch_size 16 \
    --epochs 30 \
    --lr 1e-4 \
    --backbone facebook/dinov2-base \
    --decoder unet \
    --loss_weights class_balanced
```

### Training SegFormer

```bash
python train_segformer.py \
    --data_dir ./data/desert_dataset/ \
    --batch_size 12 \
    --epochs 35 \
    --lr 6e-5 \
    --backbone nvidia/mit-b4 \
    --augment heavy
```

### Training the Unifier

```bash
python train_unifier.py \
    --dinov2_ckpt checkpoints/dinov2_best.pth \
    --segformer_ckpt checkpoints/segformer_best.pth \
    --epochs 10 \
    --lr 1e-3
```

### Training Configuration

**DINOv2 Configuration:**
- Optimizer: AdamW (lr=1e-4, weight_decay=0.01)
- Loss: 0.5 × Dice + 0.5 × Weighted Cross-Entropy
- Augmentations: Optical Distortion, Grid Distortion
- Batch Size: 16

**SegFormer Configuration:**
- Optimizer: AdamW (lr=6e-5, weight_decay=0.01)
- Loss: 0.4 × Dice + 0.6 × Focal Loss
- Augmentations: Brightness/Contrast, Hue/Saturation shifts
- Batch Size: 12

## 🔬 Technical Approach

### 1. Multi-Scale Feature Fusion

**DINOv2 Decoder:**
```
F_out = Conv3×3(Upsample(Concat(F_high, F_low)))
```
Fuses features from transformer layers {2, 5, 8, 11} for sharp boundaries.

**SegFormer Decoder:**
```
F_fused = Linear(4C, C)(Concat(F₁ᵘᵖ, F₂ᵘᵖ, F₃ᵘᵖ, F₄ᵘᵖ))
```
All-MLP decoder aggregates pyramid features at {1/4, 1/8, 1/16, 1/32} resolutions.

### 2. Ensemble Fusion Strategy

```
P_final = Conv1×1(Concat(Logits_SegFormer, Logits_DINOv2))
```

The learnable unifier acts as per-pixel weighted voting, automatically discovering:
- Higher SegFormer weight for "Logs" (fine detail)
- Higher DINOv2 weight for "Sky" (global context)

### 3. Loss Function Design

```python
total_loss = α × dice_loss + β × weighted_ce_loss

# Class weights computed as:
weight_i = 1 / log(frequency_i + ε)
```

This combats extreme class imbalance while preserving boundary sharpness.

## 🛠️ Challenges & Solutions

### Challenge 1: Ambiguous Ground Truth Labels

**Problem:** Ground Truth inconsistently labeled small debris as either "Ground Clutter" or "Rocks"

**Solution:** Post-processing class remapping
```python
# Merge semantically similar classes
prediction[prediction == CLUTTER_ID] = ROCK_ID
```
**Impact:** +10% mIoU improvement

---

### Challenge 2: Texture Confusion (DINOv2)

**Problem:** ViT backbone over-smoothed textures, merging "Dry Grass" and "Dry Bushes"

**Solution:** 
- Geometric augmentations (OpticalDistortion, GridDistortion)
- Increased Dice loss weight (0.4 → 0.5)

**Impact:** Sharpened vegetation boundaries by 15%

---

### Challenge 3: Shadow Hallucinations (SegFormer)

**Problem:** Deep shadows misclassified as "Logs" due to intensity overfitting

**Solution:**
- Aggressive photometric augmentation
- RandomBrightnessContrast (p=0.5)
- HueSaturationValue shifts (p=0.3)

**Impact:** -15% false positive rate on shadowed regions

---

### Challenge 4: "Flower" Hallucination in Test Set

**Problem:** Model predicted flowers in barren desert (domain shift)

**Solution:** Biome-specific constraint
```python
# Mask out impossible classes for desert biome
logits[:, FLOWER_CLASS] = -float('inf')
```
**Impact:** Zero false positives for Class 6

---

### Challenge 5: Extreme Class Imbalance

**Problem:** Sky + Landscape dominate 70%+ of pixels

**Solution:** 
- Logarithmic class weighting
- Focal loss (γ=2) to focus on hard examples
- Hard example mining during training
